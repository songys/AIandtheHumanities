# 나만의 포트폴리오 만들기

GitHub Pages를 이용해 무료로 자기소개 웹사이트를 만드는 방법을 안내합니다.

코딩을 몰라도 괜찮습니다. 이 폴더의 파일을 복사해서 내용만 바꾸면 됩니다.

---

## 전체 흐름

```
1. GitHub 가입  →  2. 저장소 만들기  →  3. 파일 올리기  →  4. 내용 바꾸기  →  5. 완성!
```

---

## Step 1. GitHub 가입하기

1. [github.com](https://github.com)에 접속합니다
2. 오른쪽 위의 **Sign up** 버튼을 클릭합니다
3. 이메일 주소를 입력합니다 (학교 메일 권장)
4. 비밀번호를 설정합니다
5. **사용자 이름(username)** 을 정합니다

> **사용자 이름이 곧 내 웹사이트 주소가 됩니다!**
>
> 예: 사용자 이름이 `hong-gildong`이면
> 내 포트폴리오 주소는 `https://hong-gildong.github.io`

6. 이메일 인증을 완료하면 가입 끝!

### username 정하기 팁

- 본명 또는 본명의 변형을 추천합니다
- 영어 소문자와 하이픈(-)만 사용 가능
- 예: `gildong-hong`, `youngsook-s`, `minjae-kim`

---

## Step 2. 저장소(Repository) 만들기

저장소는 파일을 보관하는 폴더라고 생각하면 됩니다.

1. 로그인한 상태에서 오른쪽 위 **+** 버튼 → **New repository** 클릭
2. **Repository name**에 `username.github.io` 입력
   - 반드시 본인의 사용자 이름을 넣어야 합니다
   - 예: 사용자 이름이 `hong-gildong`이면 → `hong-gildong.github.io`
3. **Public** 선택 (공개)
4. **Add a README file** 체크
5. **Create repository** 클릭

---

## Step 3. 이 템플릿 파일 올리기

이 폴더에 있는 파일을 내 저장소에 올립니다.

### 방법 A: 웹에서 직접 올리기 (가장 쉬움)

1. 방금 만든 저장소 페이지에 들어갑니다
2. **Add file** → **Upload files** 클릭
3. 아래 파일들을 드래그해서 올립니다:
   - `index.html`
   - `style.css`
4. 아래로 스크롤하여 **Commit changes** 클릭
5. 1~2분 후 `https://username.github.io`에 접속하면 페이지가 보입니다!

### 방법 B: 파일을 하나씩 만들기

1. 저장소 페이지에서 **Add file** → **Create new file** 클릭
2. 파일 이름에 `index.html` 입력
3. 이 폴더의 `index.html` 내용을 복사해서 붙여넣기
4. **Commit changes** 클릭
5. 같은 방법으로 `style.css`도 만들기

---

## Step 4. 내 정보로 바꾸기

`index.html` 파일을 열어서 더미 내용을 내 정보로 바꿉니다.

### 바꿀 곳 찾는 법

파일에서 `✏️` 표시가 있는 부분을 찾으세요. 바꿔야 할 곳에 모두 표시해 뒀습니다.

| 바꿀 항목 | 찾을 텍스트 | 바꿀 내용 |
|----------|-----------|----------|
| 이름 | `홍길동` | 내 이름 |
| 태그라인 | `글과 데이터 사이에서 이야기를 찾는 사람` | 나를 한 줄로 소개 |
| 전공 | `국어국문학과` | 내 전공 |
| 자기소개 | `단국대학교 국어국문학과에...` | AI로 만든 자기소개 |
| 경험 | `글쓰기 센터 튜터`, `캡스톤 프로젝트` 등 | 내 경험 |
| 기술 | `한국어 글쓰기`, `Python` 등 | 내 기술 |
| 이메일 | `gildong@example.com` | 내 이메일 |
| GitHub | `hong-gildong` | 내 GitHub 사용자 이름 |

### 수정 방법

1. 저장소에서 `index.html` 파일을 클릭
2. 오른쪽 위 **연필 아이콘** (✏️) 클릭
3. 내용을 수정
4. **Commit changes** 클릭
5. 1~2분 후 웹사이트에서 변경 확인

---

## Step 5. 완성!

`https://username.github.io`에 접속하면 나만의 포트폴리오가 보입니다.

### 확인 체크리스트

- [ ] PC 브라우저에서 접속 가능한가?
- [ ] 스마트폰에서도 잘 보이는가?
- [ ] 이름과 자기소개가 정확한가?
- [ ] 이메일 주소가 맞는가?
- [ ] 링크가 정상 작동하는가?

---

## 자주 묻는 질문

### Q: 수정했는데 웹사이트에 반영이 안 돼요
Commit 후 1~2분 정도 기다려 보세요. GitHub Pages는 반영에 약간의 시간이 걸립니다.

### Q: 페이지가 404 에러가 나요
- 저장소 이름이 정확히 `username.github.io`인지 확인하세요
- Settings → Pages에서 Source가 `main` 브랜치로 설정되어 있는지 확인하세요

### Q: 사진을 넣고 싶어요
1. 저장소에 `images` 폴더를 만들고 사진을 업로드합니다
2. `index.html`에서 프로필 이미지 부분의 주석을 해제하고 파일 이름을 입력합니다

### Q: 디자인을 바꾸고 싶어요
`style.css` 파일에서 색상과 폰트를 수정할 수 있습니다.
AI에게 "이 CSS에서 포인트 색상을 초록색으로 바꿔줘"라고 요청해 보세요.

### Q: 섹션을 추가하고 싶어요
AI에게 "이 HTML에 블로그 섹션을 추가해줘"라고 요청하면 코드를 만들어줍니다.

## 파일 구성

```
portfolio-template/
├── README.md          ← 지금 읽고 있는 안내서
├── index.html         ← 포트폴리오 메인 페이지 (이것만 올려도 됨)
├── style.css          ← 디자인 파일
└── images/            ← 사진을 넣을 폴더 (선택)
```
